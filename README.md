# 🌱 EcoTrip — Calculadora de Impacto Ambiental para Viagens

Aplicação web desenvolvida durante o Bootcamp GitHub Copilot na DIO, com o objetivo de estimular escolhas de transporte mais conscientes e sustentáveis. A calculadora estima a emissão de CO₂ gerada por diferentes meios de transporte, ajudando o usuário a tomar decisões mais responsáveis no dia a dia.

## 📋 Sobre o Projeto

O **EcoTrip** nasceu da ideia de que pequenas escolhas cotidianas têm grande impacto ambiental. Com essa ferramenta, o usuário informa sua origem, destino e o meio de transporte utilizado, e recebe uma estimativa de quanto CO₂ será emitido naquela viagem — podendo comparar alternativas e optar pelo trajeto mais sustentável.

O projeto foi desenvolvido com o auxílio do **GitHub Copilot**, explorando na prática como a IA pode acelerar e enriquecer o processo de desenvolvimento.

## ✨ Funcionalidades

- Estimativa de emissão de **CO₂** por meio de transporte
- Comparação entre diferentes opções de deslocamento
- Interface web simples, intuitiva e responsiva
- Incentivo a escolhas de mobilidade mais sustentáveis

## 🌍 Por que esse projeto importa?

O setor de transporte é um dos maiores emissores de CO₂ no mundo. Pequenas mudanças de hábito — como preferir o transporte público, a bicicleta ou dividir uma carona — podem fazer uma grande diferença ao longo do tempo. O EcoTrip quer tornar essa informação acessível e visual para qualquer pessoa.
